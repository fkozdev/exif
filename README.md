
# Exif

Python Kullanılarak yapılmış basit bir chat uygulaması.
## Kullanılan Teknolojiler

**Dil:** Python

**Modüller:** PIL , sys

  
## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/fkozdev/exif
```

Proje dizinine gidin

```bash
  cd exif
```

Kodu çalıştırın

```bash
  python3 exif.py -r <Resim.jpg(png)> 
```
